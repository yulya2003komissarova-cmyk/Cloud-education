# ☁️ Облачные технологии в обучении программированию

Электронный образовательный ресурс (ЭОР) — интерактивный сайт-презентация по теме «Применение облачных технологий при обучении программированию».

## 🚀 Публикация на GitHub Pages

### Способ 1: Через браузер (без командной строки)

1. Зайди на [github.com](https://github.com) и авторизуйся
2. Нажми **«+» → New repository**
3. Назови репозиторий, например: `cloud-education`
4. Поставь галочку **Public**
5. Нажми **Create repository**
6. На странице репозитория нажми **uploading an existing file**
7. Перетащи файл `index.html` в окно загрузки
8. Нажми **Commit changes**
9. Перейди в **Settings → Pages**
10. В разделе **Source** выбери: **Deploy from a branch**
11. Branch: **main**, папка: **/ (root)**
12. Нажми **Save**
13. Через 1–2 минуты сайт будет доступен по адресу:
    ```
    https://ТВОЙ_ЛОГИН.github.io/cloud-education/
    ```

### Способ 2: Через Git (командная строка)

```bash
git init
git add index.html README.md
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ТВОЙ_ЛОГИН/cloud-education.git
git push -u origin main
```

Затем включи GitHub Pages в настройках репозитория (шаги 9–12 выше).

## 📝 Как редактировать контент

- Нажми кнопку **«✏️ Редактировать»** на сайте — все тексты станут редактируемыми
- После правок нажми **«💾 Сохранить»** — скачается обновлённый `index.html`
- Загрузи обновлённый файл в репозиторий (замени старый)

## 🎬 Как добавить свои видео

В файле `index.html` найди все места с `VIDEO_ID_1`, `VIDEO_ID_2` и т.д.

Замени их на реальные ID видео с YouTube. Например, если ссылка на видео:
```
https://www.youtube.com/watch?v=abc123xyz
```
То ID видео = `abc123xyz`. Замени:
```
VIDEO_ID_1 → abc123xyz
```
